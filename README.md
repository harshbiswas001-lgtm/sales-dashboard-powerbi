# sales-dashboard-powerbi
# Objective
To design an interactive dashboard for business stakeholders that visualizes key performance indicators (KPIs) like Sales and Profit, helping in data-driven decision-making.
# Tools Used
- Power BI  
- Power Query (Data Cleaning)  
- Sales Dataset (CSV)

# Dataset
The dataset contains:
- Sales  
- Profit  
- Category  
- Region  
- Order Date 

# Dashboard Features
  - KPI Cards  
  - Total Sales  
  - Total Profit  

  - Bar Chart
  - Sales by Category  

- **Line Chart**  
  - Sales Trend Over Time  

- **Slicer (Filter)**  
  - Region-wise filtering
